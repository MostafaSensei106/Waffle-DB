K-nearest neighbor search.

vector is the query embedding (must match configured dimension). k is the number of nearest neighbors to return. efSearch overrides the configured ef_search for this query. Higher values = more accurate but slower. Pass 0 to use config default.

  • efSearch: Overrides the default ef_search if > 0.
  • includeMetadata: If false, avoids Disk I/O by not fetching metadata. Default is false for speed.

Returns results sorted by distance (ascending = most similar first).
