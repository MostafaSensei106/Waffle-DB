WaffleQueryBuilder

WaffleQueryBuilder class

Fluent query builder for WaffleDB vector searches.

Usage:

final results = await db
    .queryBuilder()
    .withVector(queryVector)
    .limit(20)
    .efSearch(64)
    .threshold(0.5)
    .execute();
